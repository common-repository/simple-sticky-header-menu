== Installation ==

This section describes how to install the plugin and get it working.

1. Upload simple-sticky-header-menu folder to your wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Thats All

== Frequently Asked Questions ==

= How can i make my website header sticky =

You simply have to install plugin and activate, thats all

= After Activate Still Header is not showing sticky ?=

In some website header is not define on <header> tag on that case you have to add header class or id on plugin setting page, thats all,

= How will i get my website header id or class name ? =

It's a very simple step, even you are nontechnical, just activate plugin you will see on setting page a video tutorial, how to fine css name or id.
